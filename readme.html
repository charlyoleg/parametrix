Parametrix

Presentation

A static website to generate few 2D paramteric designs.

This static website is deployed on github-pages. The sources are available on github.

Prerequisite

  • node version 18.13.0 or higher
  • npm version 9.3.0 or higher

Developing

git clone https://github.com/charlyoleg/parametrix
cd parametrix
npm i
npm run dev -- --open

Other useful commands:

npm run format
npm run lint
npm run check
npm run test:unit -- --run
npm run build
npm run preview
npm run clean
